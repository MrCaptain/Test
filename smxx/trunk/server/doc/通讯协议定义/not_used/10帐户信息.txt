########### 创建角色 ##############
协议号:10003
c >> s:
    int:16 服务器标识
    int:8 职业
    int:8 性别
    string 名字
s >> c:
    int:8
		1 => 创建成功
		0 => 失败
		2 => 未知错误
		3 => 角色名称已经被使用
		4 => 非法字符
		5 => 角色名称长度为1~5个汉字
		6 => 用户已经创建角色
		7 => 账号角色个数已经达到3个
		8 => 角色名中含有敏感词
	int:64 角色ID (创建成功时, 否则为0)

########### 被下线通知 ##############
协议号:10007
s >> c:
    int:8
	0 => 服务器端维护
	1 => 账号在别处登录
	2 => 被踢下线
	3 => 被封角色
	4 => 被封账号
	5 => 在线时间超过3小时（防沉迷）
	6 => 9级以下，30分钟都不移动, 则做踢出
	7 => 涉嫌使用外挂
